Select Desired Report

Below you can choose what kind of data you'd like to view. Each view will have a number of graphs you can swap between to visuals the data to your preference There are 3 general views

  • General Date Report - This will show a breakdown of the data date by date, showing how many appointments were taken on each day
  • General Symptom Report - This will show a breakdown of the most common symptoms and the number of times they appear in the data
  • General Diagnosis Report - This will show a breakdown of the most common diagnosis and the number of times they appear in the data

    • Furthermore there are 2 specific views that allow a deep dive into a particular data point - these are the
  • Specific Symptom Report
  • Specific Diagnosis Report
    • These reports will allow you to single out a specific symptom or diagnosis to further study, showing the number of times it has occurred on what days.

    Furthermore you can also refine the their chosen report by choose a date range, when inputted the reports will only generate using data that falls between the start and end date

    When entering dates please make sure to use the "YYYY-MM-DD" format
